# Clean Calculator

A clean, intuitive calculator with modern UI that performs basic arithmetic operations (addition, subtraction, multiplication, division). Features robust error handling for edge cases like division by zero, full keyboard support, and responsive design.

![Calculator Screenshot](https://via.placeholder.com/800x450/6a11cb/ffffff?text=Clean+Calculator+Screenshot)

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Error Handling**: Proper handling of division by zero and overflow conditions
- **Keyboard Support**: Full keyboard input with intuitive shortcuts
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Clean UI**: Modern gradient design with intuitive button layout
- **Calculation History**: Shows the full calculation process
- **Multiple Clear Options**: Clear (C) and Clear Entry (CE) functionality

## Keyboard Shortcuts

| Key | Function |
|-----|----------|
| `0-9` | Number input |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `.` | Decimal point |
| `Enter` or `=` | Calculate result |
| `Escape` | Clear all |
| `Backspace` | Delete last digit |
| `C` | Clear entry |
